# Usage

## Build images

The following script increments version and relase information, as well as, builds each image based on the target source directory.

### CentOS (base)

```
sh bin/init.sh cos
```

### MongoDB Enterprise

```
sh bin/init.sh mdb
```

### MongoDB Ops Manager

```
sh bin/init.sh opm
```
